General Rules

  • A valid relation must be either an equation, an inequality, or a compound formula, such as y = sin(x) or sin(x) < cos(y).

  • Enter the product of x and y as x y, instead of xy.

    • You don’t need a space around parentheses or in a few situations such as 2x .

  • Enter the sine of \(x\) as sin(x), instead of sinx or sin x.

  • Use parentheses to group a part of the relation like 1 / (x + 3).

  • Each point in Cartesian coordinates has infinitely many identities in polar coordinates. For example, \((x, y) = (1, 0)\) is identical to \((r, θ) = (1, 0), (1, 2π), (1, −2π), (1, 4π), …; (−1, π), (−1, −π), (−1, 3π), …\)

    • You can restrict the range of r or θ as in r = θ ∧ 0 ≤ θ ≤ 2π.

  • You can make a parametric relation by using a real parameter t as in x = cos(t) ∧ y = sin(t), x + i y = exp(i t) or x = t cos(t) ∧ y = t sin(t) ∧ 0 ≤ t ≤ 2π.

    • Any relation that contains t is implicitly interpreted as \(∃t ∈ 𝐑[…]\)

MFS

There are different ways of defining exponentiation for zero and negative exponents. The following definition is used in Graphest.

  • For any coprime integers \(p\) and \(q\) \((q > 0)\), \(x^{p/q}\) is defined to be \((\sqrt[q]{x})^p\), where \(\sqrt[q]{x}\) is the real, non-negative \(q\)th root of \(x\).

    • \(x^{p/q}\) is an even (odd) function of \(x\) if \(p\) is even (odd).

    • For \(x < 0\), \(x^y\) is defined if and only if \(y\) is a rational number with an odd denominator.

  • \(0^0\) is defined to be 1.
